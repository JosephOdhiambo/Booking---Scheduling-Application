Welcome to our Bus Reservation System!

Bus Scheduling

As soon as you make a booking on the application, you are automatically scheduled to the departure time of a specific bus

Create new buses

The system is able to store data about buses. As soon as new bus is created by you as the admin, the end user is able to make booking on his/her end

Reporting of data

The administrator has the ability to print out reports made from data within the system. This will help in future planning
